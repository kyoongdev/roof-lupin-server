# 루프루팡

## Introduction

옥상 공간 중개 프로젝트입니다. 공간주(호스트)가 호스트 페이지에서 옥상 공간을 등록하면 사용자가 해당 공간을 예약 혹은 즉시 결제가 가능합니다.

백엔드의 기능 개발을 담당하였으며, 배포 환경 구성 또한 진행하였습니다.

## Stacks

- Nest.js
- TypeScript
- Prisma
- MySQL
- AWS EC2
- AWS Aurora

## Role

- BackEnd : 포스페이먼츠 결제 모듈 연동, API 개발, 데이터베이스 설계를 담당하였습니다.
